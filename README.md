# Proyecto SaborPlus
Aplicación web integral para la gestión, optimización y control de restaurantes.

## Descripción
SaborPlus es un desarrollo realizado en el marco del programa "Análisis y Desarrollo de Software" del SENA. El sistema busca automatizar los procesos operativos, desde la toma de pedidos hasta el control de inventarios.

## Tecnologías utilizadas
- Git/GitHub (Control de versiones)
- HTML/CSS/JavaScript (Frontend)
- Visual Studio Code

## Estado actual
Actualmente, estoy trabajando en la estructura base y la documentación técnica de los componentes del sistema.
